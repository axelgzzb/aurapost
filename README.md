# aurapost
Aurapost is a full-stack social network simulation developed as part of the Interactive Communication degree at UAB. The project focuses on user interaction, secure authentication, and real-time content management using a PHP and MySQL architecture.

The platform allows users to create profiles, share posts, and interact within a community-driven feed. The project emphasizes back-end logic, data persistence, and a clean UI/UX.
